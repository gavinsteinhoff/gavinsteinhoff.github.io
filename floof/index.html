Floof

Floof is a simple javascript library for generating sample text.

You can easily add random text
<p class="floof"><p>

You can easily add styled text
<p class="floof floof-x"><p>

You can easily add long text
<p class="floof floof-lg"><p>

  • Every Floof Option
  • "floof" - default, must include
  • "floof-sm" - only one paragraph of floof
  • "floof-md" - a few paragraphs of floof
  • "floof-lg" - many paragraphs of floof
  • "floof-x" - floof with bold and links, can be combined with sizes
